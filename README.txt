Mini Sculpts Studio — динамические QR на статическом хостинге
Дата: 2025-08-15

Что это
-------
Готовый «скелет» сайта, который позволяет делать динамические QR без сервера:
QR указывает на /m/КОД, а редирект на нужную сцену берётся из файла /m/map.json.

Как развернуть (GitHub Pages, бесплатно)
----------------------------------------
1) Создай аккаунт GitHub и новый репозиторий, например minisculpts-ar.
2) Загрузите ВСЕ файлы из этого архива в репозиторий.
3) В настройках репо включите GitHub Pages (Settings → Pages): Source = Deploy from a branch; Branch = main; Folder = /root.
4) Твой сайт будет по адресу: https://<твой_юзер>.github.io/minisculpts-ar

Как формировать ссылки для QR
-----------------------------
Формат коротких ссылок для печати:
  https://<твой_юзер>.github.io/minisculpts-ar/m/HB01

Файл соответствий /m/map.json (пример):
{
  "HB01": "/ar/index.html?v=1&scene=hbday_cats&msg=%D0%A1%20%D0%B4%D0%BD%D1%91%D0%BC%20%D1%80%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D1%8F!&palette=party&n=001",
  "HB02": "/ar/index.html?v=1&scene=hbday_monkeys&msg=%D0%A1%20%D0%BF%D1%80%D0%B0%D0%B7%D0%B4%D0%BD%D0%B8%D0%BA%D0%BE%D0%BC!&palette=gold&n=002",
  "GZ01": "/ar/index.html?v=1&scene=congrats_fireworks&msg=%D0%9F%D0%BE%D0%B7%D0%B4%D1%80%D0%B0%D0%B2%D0%BB%D1%8F%D1%8E!&palette=neon&n=101"
}

Значение может быть либо абсолютным URL, либо относительным путём/строкой параметров.
Если значение начинается с "http", будет редирект на внешний адрес.
Иначе произойдёт редирект на /ar/index.html + эта строка.

Как добавлять новые коды
------------------------
- Открой /m/map.json, добавь новую пару "КОД": "значение", закоммить в GitHub.
- Готово: QR вида /m/НОВЫЙКОД начнёт вести на нужную сцену.

Где печатать QR
---------------
- Размер 18–22 мм, ч/б, коррекция H, матовая плёнка (PP/BOPP + мат-лам).
- Размещай снизу базы и на карточке-вкладыше.

План на будущее
---------------
- В папке /ar/index.html подключить WebAR-движок и подставлять scene/msg/palette из URL.
- Когда появится свой домен — просто подключи его к GitHub Pages; структура ссылок сохраняется.
